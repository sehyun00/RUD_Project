ocr.yaml을 이용해 아나콘다 가상환경 생성(gpu 환경 기준)

사용시 순서 지키셈
1. 가상환경(conda) 파이썬 3.7 생성 후 실행

2. pipenv 설치하고 pipenv 버전 3.7로 지정
-> pip install pipenv
-> pipenv --python 3.7

3. pororo 설치하고

4. 그다음에 opencv, mataplot 등등 설치
-> pip install opencv-python~=4.8.0.74
-> pip install wget
-> pip3 install torch torchvision torchaudio
-> pip install matplotlib

5. 이제 easyocr 설치
-> 아나콘다 프롬프트에서
-> conda install easyocr

5-1. 만약 안된다면
컴퓨터에 c++이 없어서 그럼
그래서 visual studio 설치하고
rustc를 설치해야 easyocr을 설치할 수 있음

6. 플라스크 설치
-> pip install flask

cuda 버전(gtx 1650, cuda 10.1)
pip install torch==1.7.1+cu101 torchvision==0.8.2+cu101 torchaudio==0.7.2 -f https://download.pytorch.org/whl/torch_stable.html
